directive: chartColors

chartColors

chart-colors is used to specify the colors to use in the chart. You can provide the colors or a function to determine the colors.

Restrict To: Element

Parent Element: c3chart

Attributes:
Name Type Description
color-pattern String

A string containing comma separated hex colors

c3js docs
color-function function

Provide a function that receives the value to determine a color for that value.

c3js docs
Example
Usage:
  <chart-color color-pattern="..." color-function="..."/>
